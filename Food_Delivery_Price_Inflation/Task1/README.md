# Food Delivery Price Inflation Analysis

## Project Overview
This project analyzes price differences between restaurant dine-in prices and online food delivery app prices to identify price inflation and misleading discounts.

## Dataset
The dataset contains restaurant name, food item, dine-in price, app price, discount shown, delivery platform, city, and cuisine type.

## Task 1: Data Cleaning & Preparation
- Handled file encoding issues
- Standardized column names
- Removed duplicate records
- Removed invalid prices
- Created price inflation metrics
- Generated an analysis-ready dataset
